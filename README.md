# Трекер прочитанных книг

Консольное приложение для учёта прочитанных книг.

## Запуск
1. Установите Python 3.8+.
2. Выполните: `python main.py`

## Возможности
- Добавление книги (автор, название, оценка 1–5, дата)
- Просмотр всех книг
- Расчёт средней оценки
- Статистика по авторам
- Удаление книги
- Данные сохраняются в books.json
